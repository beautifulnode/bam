# BAM

The easiest static site generator on the planet.

# Requirements

* [nodejs](http://nodejs.org)

## Install

``` sh
npm install bam -g
```

## Create a new site

``` sh
bam new foo-site
```

## Run in dev mode

``` sh
cd foo-site
bam run
```

## Generate site

``` sh
bam gen
```
